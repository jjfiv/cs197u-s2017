These are the commands I know

  • cd - changes the current working directory. wihout commands, goes to home directory
  • mkdir - give user the option to create a directory with a given name
  • ls - list files in working directory, or given directory
  • git status - displays the current state of the working directory
  • git commit - records changes to the repository
  • git push - updates repository at remote location
  • git pull - updates repository on local machine
  • ls -la - displays all files, even hidden files
  • touch filename - creates a file with given name
  • mkdir directory_name - creates a directory with given name
  • rm -r file_name - removes file without confirmation
