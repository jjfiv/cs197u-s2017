These are the commands I know

  • cd
    • Change the directory you are in.
  • mkdir
    • Make a new directory.
  • ls
    • Prints the contents of the current directory to standard output.
  • git status
    • Prints the status of all the files in the local respository to standard output.
  • git commit
    • Adds a new commit, or save point, to the log of commits on the respoitory.
  • git push
    • Updates the repository on the server with the contents of your local repository.
  • git pull
    • Updates the local respository with the contents of the repository on the server.
  • git checkout Allows a user to create a new branch in git and then write to that branch instead of overriding the work in the master branch.
  • git reset --hard Allows a user to rewind in time and set the head commit as a previous commit to reset the repository to that state
  • touch Allows a user to create a new file
  • pwd Prints the current directory to standard output
