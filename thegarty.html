These are the commands I know

  • cd - change directory
  • mkdir - make directory
  • ls - list current directory items
  • git status - show the current branch
  • git commit - commit changes made to local repository copy with a message
  • git push - push committed changes to git repository
  • git pull - pull recent changes from git repository
  • rm - remove a file
  • grep - search a file
  • git checkout - change branches in git
  • chmod - change access permissions to file system objects
